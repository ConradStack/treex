NAME

Treex::Tool::Depfix::CS::FormGenerator

DESCRIPTION

This package provides the get_form method, which tries to generate the wordform corresponding to the given lemma and tag.

METHODS

my $form = $formGenerator->get_form($lemma, $tag)

Returns the form corresponding to the given lemma and tag, or undef if no form can be generated. In such case, it also issues the following warning: "Can't find a word for lemma '$lemma' and tag '$tag'."
